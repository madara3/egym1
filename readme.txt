	Requirements:
	=========

	- QT Creator 4.2.0
	- QT 5.7.1 framework
	- Windows 8.1 x64


	Folders:
	=====
	
	debug - contains debug build of the demo + required debug dlls
	release - contains release build of the demo + required dlls
	res - contains resources used by the app


	How to build:
	=========
	
	open egym.ro in QT Creator the run QMake and build
	disable shadow build in project settings
